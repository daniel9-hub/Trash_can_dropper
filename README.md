# Trash_can_dropper
This is a game about trying to drop trash into a trash can
# How to play
- To drop trash, press the space bar
- Every time you hit the trash can, you get more score and you also move faster by a random amount
- When you miss, you can try again
- To run the game, just run trashcandropper.exe
